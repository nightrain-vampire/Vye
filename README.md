# Vye

云E办项目，适合小白练手。采用SpringBoot+Vue前后端分离式开发。

#### 预期实现模块

- 登录

- 职位管理

- 职称管理

- 部门管理

- 操作员

- 员工管理

- 工资管理

- 个人中心

- 在线聊天

> 能做多少算多少 orz

#### 技术栈

- Vue3

- ElementUI

- SpringBoot

- MySQL

- MyBatis-Plus-Generator

#### 自动化代码生成

使用MyBatis-Plus的代码生成器，可连接数据库自动生成pojo，mapper，service，controller四层的大部分代码，使得注意力可集中于开发业务逻辑上。
